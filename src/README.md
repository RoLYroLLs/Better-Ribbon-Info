# Better Ribbon Info

Sid Meier's Civilization VII mod to show more information on the player ribbon.

**What Is It?**

Adds better ribbon information to the game. Includes the following data:
- ![Production](./assets/icons/bri-icon-production-10x10.png) Production
- ![Citizens](./assets/icons/bri-icon-citizens-10x10.png) Citizens
- ![Combat Power](./assets/icons/bri-icon-combat_power-10x10.png) Combat Power
- ![Food](./assets/icons/bri-icon-food-10x10.png) Food

Includes the ability to turn on and off the visibility of each resource.

**Saved Games**

This mod does not affect saved games.

**Notes**

This mod is derived from 3 of [izica's](https://steamcommunity.com/profiles/76561198026145429/myworkshopfiles/?appid=1295660) mods: [Player production](https://steamcommunity.com/sharedfiles/filedetails/?id=3510517981), [Player population / citizens](https://steamcommunity.com/sharedfiles/filedetails/?id=3510510162), [Player combat power](https://steamcommunity.com/sharedfiles/filedetails/?id=3506950841) on Steam.
