# Changelog

All notable changes to this project will be documented in this file.

## [1.1.1] - 2026-01-08

### Added

- Ability to see players' Food in the Ribbon
- Ability to add or remove Food from list

- Changed the icon for Combat Power

- initial release

### Added

- Ability to see players' Production in the Ribbon
- Ability to add or remove Production from list
- Ability to see players' Citizens in the Ribbon
- Ability to add or remove Citizens from list
- Ability to see players' Combat Power in the Ribbon
- Ability to add or remove Combat Power from list
