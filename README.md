# Better Ribbon Info
Sid Meier's Civilization VII mod to show more information on the player ribbon.

This is a single mod which includes 3 of [izica's](https://steamcommunity.com/profiles/76561198026145429/myworkshopfiles/?appid=1295660) mods:
- [Player production](https://steamcommunity.com/sharedfiles/filedetails/?id=3510517981)
- [Player population / citizens](https://steamcommunity.com/sharedfiles/filedetails/?id=3510510162)
- [Player combat power](https://steamcommunity.com/sharedfiles/filedetails/?id=3506950841)
